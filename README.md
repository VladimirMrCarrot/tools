# tools

Мій маленький хаб навколо **student dev tools**: як оформити студентські підписки, що з них витиснути й як навколо цього зібрати свою інфру та схемки.

---

## Що тут взагалі відбувається

У цьому репо я тримаю штуки, які часто потрібні «на памʼять»:

- Гайд по **GitHub Student Developer Pack** і основних сервісах навколо нього.
- Нотатки про інфру: домени, сервіси, діаграми, типові сетапи.
- Інтерактивну шпаргалку по **типам звʼязків між сутностями** (щоб не вигадувати заново на кожній діаграмі).
- Допоміжні файли для AI‑тулінгу та PlantUML.

По факту це мій кишеньковий довідник, щоб не збирати цей контекст кожного разу з нуля.

## Student Pack гайд

Папка `student-pack/` — це мій **how‑to** по GitHub Student Developer Pack:

- як його отримати;
- які саме сервіси дають профіт (Miro, Figma, JetBrains, ElevenLabs, Google AI і т.д.);
- на що звернути увагу з термінами підписок та автопродовженням;
- як це все підʼєднати до своїх реальних проєктів, а не просто «зареєстрував і забув».

Live‑версія: `https://tools.mrcarrot13666.me/student-pack/`

---

## Interactive connection types

Папка `conection_types/` + відповідна сторінка — це:

- інтерактивна підказка по **видам звʼязків між сутностями**;
- корисно, коли малюєш діаграми (ERD, UML, інфра‑схеми) і треба швидко згадати, який тип звʼязку логічно поставити;
- зручна штука, щоб тримати консистентний підхід у всіх своїх схемах.

Live‑версія: `https://tools.mrcarrot13666.me/conection_types/`
